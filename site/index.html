My Azure Static Website

Project: Static site on Azure

This is a minimal placeholder page for my resume project. It demonstrates static hosting on Azure Blob Storage.

About This Project

This project demonstrates my ability to deploy and manage Azure resources for cloud-based web hosting. It was built as part of my cloud engineering learning path and showcases automation, scalability, and cost-effective architecture.

Actions performed in this project

  • Created and configured Azure Resource Group and Storage Account (StorageV2)
  • Deployed static website files to Azure Blob Storage using az storage blob upload-batch
  • Implemented CI/CD pipeline using GitHub Actions for automatic deployment
  • Optional extension: custom domain and HTTPS using Azure Static Web Apps

How to deploy (quick)

az storage blob upload-batch -s ./site -d '$web' --account-name mystaticsite01 --auth-mode login
